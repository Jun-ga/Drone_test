## 파일 실행 관련 공지
* 실제 대회 당일에는 각 팀의 `Gihub Repository`에서 코드를 `Clone`하여 비행시킬 예정
* 대회에서 실행하고자 코드의 파일명을 다음으로 통일
  * `main.m`
* 팀 등록시 설정한 `팀 Github Repository`에 실행파일 업로드
  * 해당 파일이 없을시, __기권__ 으로 간주
  * 코드 실행이 안될시, __미션 실패__ 로 간주
